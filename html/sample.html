Scripting

Scripting can be done inside script tag. This will change the content of an element with random words using the script below. The script is triggered by pressing the button which will an onClick event defined.

<p>The word that will come out: <span id="theWord"></span></p>
<button onClick="changeWord()" type="button" name="button">Click to Change Title</button>
<script type="text/javascript">
function changeWord(){
let r = Math.random().toString(36).substring(2,8);
document.getElementById('theWord').innerHTML = r;
}
</script>
